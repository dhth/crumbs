# Crumbs

This directory documents `crumbs`, a local CLI and TUI for surfacing progress
from coding agents.

The docs are intended to be useful both to humans reading the repo and to AI
agents that need a compact source of truth for the tool's behavior.

## Document Guide

- [`Intro`](./intro.md) - tool purpose, initial scope, and key decisions
- [`CLI`](./cli.md) - agent-facing commands and expected command behavior
- [`Session Lifecycle`](./session-lifecycle.md) - session state model
- [`TUI`](./tui.md) - default TUI layout and interaction expectations
- [`Storage`](./storage.md) - persistence model and database choices

As implementation evolves, these docs should stay lightweight and concrete.
